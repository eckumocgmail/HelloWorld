Запрос JSON-объекта по заданному url-адресу

url:
  • При выполнении запроса форма посылает знание введенное в тестовое поле в функция 'app.getInstance().submitEvent( url.value )'
  • 'app.getInstance().submitEvent( url.value )' с полученным значением URL-адреса и 'app.addObject( url, callbackFunc )'
  • Функция 'app.request( url )' выполняет обработку ошибок на стороне сервера (404, 403, 500 и т.п.), которые передаются в функцию 'app.error()'.
  • В случае успешного выполняет функция 'app.request( url )' проверяет значение 'contentType' ответного сообщения.
  • При получении валидного JSON объекта он должен он преобразовывается в cоответствующий ему HTML-элемент и передавается в качестве аргумента функции 'callbackFunc'.
  • Одновременное выполнение нескольких запросов по одному адресу запрещено, в случае такой ситуации новый запрос становиться в очередь и выполняться после завершения предыдущего запроса (как в случае успешного выполнения так и в случае прерывания с ошибкой).
  • Все вспомогательные функции являются частью объекта 'app'.
